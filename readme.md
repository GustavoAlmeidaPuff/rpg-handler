## Instalação

1. Clone o repositório:
   ```bash
   git clone https://github.com/GustavoAlmeidaPuff/rpg-handler
   ```

2. Navegue até o diretório do projeto:
   ```bash
   cd rpg handler
   ```

3. Instale as dependências:
   ```bash
   npm install
   ```

4. Inicie o servidor de desenvolvimento:
   ```bash
   npm start
   ```
